# 🤖 AI Interview & Resume Assistant

An AI-powered web application that helps job seekers prepare for interviews, analyze their resumes, identify skill gaps, and generate ATS-friendly resumes tailored to specific job descriptions.

Built with the **MERN Stack** and powered by the **Google Gemini API**, the platform combines interview preparation and resume optimization into one seamless experience.

---

## 🚀 Features

### 📄 Resume Upload

* Upload your resume in PDF format.
* Extracts and analyzes resume content.

### 💼 Job Description Analysis

* Paste any job description.
* AI analyzes the required skills, responsibilities, and qualifications.

### 👤 Self Description

* Add your professional summary or personal introduction.
* Helps the AI personalize interview questions and resume generation.

### 💻 Technical Interview Questions

Generates role-specific technical interview questions based on:

* Resume
* Job Description
* User Background

### 🤝 Behavioral Interview Questions

Creates personalized HR and behavioral interview questions to help users prepare for real interview scenarios.

### 📊 Compatibility Score

Calculates how well the user's profile matches the target job.

Includes:

* Overall Match Score
* Strengths
* Areas for Improvement

### 📈 Skill Gap Analysis

Identifies:

* Missing Skills
* Skills to Improve
* Recommended Technologies
* Learning Suggestions

### 📝 AI Resume Generator

Generates an ATS-friendly resume specifically optimized for the provided job description.

# ⚙️ Installation

## Clone the Repository

```bash
git clone https://github.com/yourusername/your-repository.git
```

```bash
cd your-repository
```

## Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

## Environment Variables

Create a `.env` file inside the server folder.

```env
PORT=5000
MONGODB_URI=YOUR_MONGODB_URI
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

## Run the Project

### Backend

```bash
npm run dev
```

### Frontend

```bash
npm start
```
